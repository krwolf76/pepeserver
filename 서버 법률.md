1. 연합(동맹) 이란..?
   1.1 연합(동맹) 은 본인 클랜 과 다른 클랜 과 힘을 합쳐서 **레이드** 또는 **컴뱃** 하는 행위를 뜻한다.
   1.2 서버는 클랜을 자동으로 가입을 하도록 설정 되어있습니다.
      * 클랜을 탈퇴하거나 삭제 할 경우 **경고 알림** 이 표시 됩니다.
      * 클랜을 60초 안에 생성하거나, 가입하지 않을 경우 서버에서 **1시간 밴** 이 되도록 설정 되어있습니다.
      * 클랜을 탈퇴 하거나, 삭제 됫을 때 경고 알림은 인게임 플레이 하는대 방해 되도록 설정 되어있습니다. **[고의]**
      
   1.3 서버는 클랜을 중점으로 연합(동맹) 관계 인지 확인합니다.
   
2. 연합 제재 설명
   2.1 연합(동맹) 을 할 경우 소규모 로 플레이하는 팀 또는 혼자서 하는 플레이어는 힘들 수 바깨없습니다. 그렇기에 제재를 하여, 밸런스를 맞추기 위함        입니다.
   2.2
   
3. 연합이 해당 되는지.
   * 해당됨.
      1. 본인 클랜 과 다른 클랜이 힘을 합쳐서 레이드, 컴뱃 하는 경우
      2. 본인 클랜 과 다른 클랜이 함깨 있을 경우
      3. 본인 클랜 과 다른 클랜이 아이템을 주고 받는 경우 
       **( 교환 및 거래 이(가) 필요한 경우는 /trade <플레이어> 또는 안전 지역 또는 자유 광장 을 참고해주세요 )**
      4. 본인 클랜 과 다른 클랜이 서로 안죽이는 경우 **( 실력 차이 일 경우 패스 )**
      5. 레이드 블럭, 컴뱃 블럭 지역일 경우
      
   * 해당되지 않음.
      1. 자유 광장 에 있을 경우
      2. 안전 지역 에 있을 경우
      3. 피아노 등 악기 가 있는곳에서 싸우지 않고 노는 경우 **( 왠만 해선 자유 광장 이용 바람. )**
      
      
      
4. 연합 제재 **( 밴 또는 행동 제약 )**

### 비속어, 비하 그외 사용 제재 법
* 서버 내에 자동 으로 욕설을 필터 및 채팅 금지 합니다.
* 비속어, 비하 그외 발언을 필터 안되도록 하거나, 글자를 변경하여 욕설을 할 경우 최소 **1일** 이상의 처벌 이 됩니다.
    * ex) 시발놈아 -> 슁봘놈아 이런식.

* 타인을 존중 해주세요.
* 패드립(부모욕설) 를 할 경우 최소 **3일** 이상의 처벌 됩니다.
* 여성 비하, 남성 비하 발언, 국가 비하 발언(일본,중국은 가능), 외모 비하 발언 시 **1일** 이상 처벌 됩니다.
* 성 비하 할 경우 **3일** 이상 처벌됩니다. (**신고자가 고소를 원할 경우 채팅 기록을 줄 수 있습니다.**)
* 음성 으로 욕설, 비하 할 경우 채팅금지+보이스 차단 **7일** 이상 됩니다.
    * 음성 신고 방법은 반드시 **증거물 영상**이 필요합니다.
    
* 비리 무새는 **영구밴** 대상입니다. 
      * 특별한 이유 와 설명을 하고 싶거나, 듣고 싶으면 디스코드: KR_WOLF#5912 친추 하시고 문의를 해주세요.
* 디도스 를 할 경우 **영구 밴** 대상 입니다. **( 디도스를 할줄 아는사람도 밴 대상입니다. )**
